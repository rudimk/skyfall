Skyfall

Skyfall was the last iteration we worked on, while running MathHarbor as a commercial product. It was essentially the backend that would help users start sandboxed IPython kernels for mathematical and scientific computing.

Current status

The code is extremely buggy, and probably doesn't work anymore. It will need significant rewrites to get anywhere. The project will probably be abandoned in favor of a new codebase, preferably in Javascript. Skyfall's just been put up here out in the open, and for all you know, some part of the codebase might help you somewhere.
